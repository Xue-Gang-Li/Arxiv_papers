        Space-time tradeoff in networked virtual distillation

        Tenzan Araki, Joseph F. Goodwin, Bálint Koczor
        摘要：这篇文章主要介绍了模块化的网络量子架构，这种架构通过使用量子通信链接将较小的量子硬件节点进行互联，提供了一种扩展性强的方法。虚拟蒸馏（VD）是一种在理想条件下可以随着量子状态副本数量增加而使错误指数级被抑制的技术。然而，实现虚拟蒸馏所需的额外的门操作会引入更多的错误，这可能会限制其实际有效性。在这项工作中，我们分析了三种将虚拟蒸馏付诸实践的方法，这些方法对应于最大化空间-时间权衡的极端情况。具体来说，我们考虑了一种尽量减少qubit数量但引入更深度量子电路的实现，并与通过使用额外的qubit来并行准备副本的实现进行了对比，其中包括一种恒定深度的实现。我们严格地描述了它们的电路深度和门操作数需求，并为在网络量子系统中实施这些方法开发了明确的架构——同时还详细说明了它们在早期容错纠错量子架构中的实现。在现实世界离子阱系统的噪声特性下，我们对这三种实现的性能进行了数值比较，并得出了以下结论。首先，即使对于非常嘈杂的状态，虚拟蒸馏也能有效抑制错误。其次，恒定深度的实现在性能上始终优于尽量减少qubit数量的实现。最后，这种方法对远程纠缠操作中的错误非常强健，其性能主要受到局部门操作噪声的限制。

          Highly efficient microwave memory using a superconducting artificial chiral atom

          Kai-I Chu, Yung-Fu Chen, Wen-Te Liao
          摘要：A microwave memory using a superconducting artificial chiral atom embedded in a one-dimensional open transmission line is theoretically investigated. By applying a coupling field to a single artificial atom, we modify its dispersion, resulting in a slow probe pulse similar to electromagnetically induced transparency.                                         Noise-strength-adapted approximate quantum codes inspired by machine learning

                                        Shuwei Liu, Shiyu Zhou, Zi-Wen Liu, Jinmin Yi
                                        摘要：我们证明了机器学习是一种强大的工具，可以用于发现超越传统代数框架的新的近似量子纠错（AQEC）代码。基于混合量子-经典学习的直接观察，我们发现了两种具有创新的噪声强度自适应（NSA）特性的新型4-qubit幅度 damping代码，其中编码字会随着噪声强度变化。它们分别是NSA自补码和NSA对补码。我们展示了它们在处理幅度 damping噪声方面优于传统代码。4-qubit自补NSA代码在保真度和Knill-Laflamme条件违反方面优于标准的LNCY AD代码。对补代码（没有已知的非NSA类似物）通过更高阶的损耗抑制和更好的保真度实现了更好的性能。我们进一步将这两种方法推广到任意系统规模的NSA AD代码家族，以及单光子损耗的0-2-4二项式代码的NSA变体。我们的结果表明，针对噪声强度的自适应可以系统地带来显著的纠错能力改进，并展示了机器学习如何帮助发现传统设计方法可能无法产生的新型有价值的代码形式。
