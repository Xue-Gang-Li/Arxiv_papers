                                  Engineering CSS surgery: compiling any CNOT in any code

                                  Clément Poirson, Joschka Roffe, Robert I. Booth
                                  摘要：我们介绍了一种在CSS量子纠错码中实现逻辑操作的框架，该框架基于Cowtan和Burton [CB24]的外科手术方法。我们的方法提供了一个系统化的方法论，用于设计和分析外科手术协议。在物理层面，我们引入了子码的概念，其中封装了执行手术所需的所有必要数据。在逻辑层面，利用同伦代数，子码使我们能够跟踪由任何手术协议引起的逻辑操作，无论逻辑操作符基础的选择。特别是，我们对码的逻辑操作符结构不做任何假设。作为概念证明，我们开发了一种受晶格手术启发的手术协议，该协议在任何CSS码的两个逻辑量子比特之间实现了具有容错保证的逻辑CNOT门。
