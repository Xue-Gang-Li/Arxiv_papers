    Sameer Dambal, Akira Sone, Yu Zhang
    摘要：模拟开放量子系统在量子计算机上面临一个基本挑战：开放量子动力学本质上是非常数的，而量子计算机通过单位演化操作。传统方法通过将非常数过程编码到单位电路中来克服这种不匹配，但这种方法会产生大量的量子位和门开销。我们提出了一个替代视角。量子处理器本身是开放系统，内在地受到噪声影响。在纠正所有错误并使用单位逻辑量子比特和门编码非常数动力学之前，我们展示了如何利用噪声作为计算资源。我们开发了一种噪声辅助的量子算法，选择性地保留物理噪声以模拟非常数信道，从而能够高效地模拟开放量子动力学，并且对量子比特要求最小。我们的方法适用于噪声中间规模量子（NISQ）设备和未来的容错架构。通过利用内在的噪声，此方法绕过了将非常数动力学编码到单位门的需要，并放松了对物理量子比特保真度要求，从而减少了量子误差纠正的开销。这一框架将噪声从限制因素转变为资源，为开放系统的实际量子模拟开辟了新的方向。

            Real-time decoding of the gross code memory with FPGAs

            Thilo Maurer, Markus Bühler, Michael Kröner, Frank Haverkamp, Tristan Müller, Drew Vandeth, Blake R. Johnson
            摘要：We introduce a prototype FPGA decoder implementing the recently discovered Relay-BP algorithm and targeting memory experiments on the $[[144,12,12]]$ bivariate bicycle quantum low-density parity check code. The decoder is both fast and accurate, achieving a belief propagation iteration time of 24ns. It matches the logical error performance of a floating-point implementation despite using reduced precision arithmetic. This speed is sufficient for an average per cycle decoding time under $1\,\mathrm{\mu s}$ assuming circuit model error probabilities are less than $3 \times 10^{-3}$. This prototype decoder offers useful insights on the path toward decoding solutions for scalable fault-tolerant quantum computers.
